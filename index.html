Personal Projects

PyBot
Github

Discord bot built with python and discord.py. The reason for this project was to have it running on our private discord server and to be able to have all the specific function we wanted into one bot.

The bot currently has various functions:

• Join voice-channels and play music from youtube links that users provide in the text message
• Memories: Take random messages from the past and re-send them
• Elaborate points system, points can be used for several different bot-functions
• Invest points in stocks based on actual stock prices (using yahoo finance api data)
• Several functions to gamble points

I have the bot running on my raspberry pi 5 that I use as a server.

LightLinks
Github

LightLinks is a webbapp built with javascript, Svelte and Supabase. This project initial purpose was to make it very easy to send multiple url:s to someone. The idea was to be able to send 1 link that contained a collection of url:s without having to login to the webbapp in any way.
This project unfortunately died out since it became a feature creep when my initial ideas for the project was proven to not be as simple as I thought. All of the latest code exists in the "develop" branch.

Window manager for windows
Github

A WPF-app built with C#.
With a keyboard-shortcut you can open a small
pop-up window with a searchbox.
When you start typing in the searchbox
the program will search for active
programs running on your computer that
matches the search. You can then press
enter to open the program that best matches
the search or step down the list of
search results to select another program.
There is also functionality to add
executionpaths for you most used programs
to be able to open a program without it already running.
